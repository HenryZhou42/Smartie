# Smartie Roadmap

## Shipped in Community Edition v1.0

- Chat with multi-provider AI (Gemini, OpenAI, OpenRouter, Ollama)
- Knowledge Base with extraction, chunking, embeddings, semantic search, RAG
- Smart Memory, Tasks, Desktop Files, Plugins, Automations
- Command palette, themes, dashboard, local SQLite storage
- No login, no telemetry, no cloud

## Future editions

- **Smartie Cloud** — hosted sync, managed AI access, team features (separate product)

## Post v1.0 ideas

- Voice input
- Document collections
- Mobile clients
- Plugin marketplace (not planned for Community Edition)
